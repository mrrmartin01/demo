## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mrrmartin01/demo.git
cd demo
npm install --legacy-peer-deps
npm run dev

```


Open http://localhost:3000 with your browser to see the result.
